Interactive Conway Game of Life

Created by Drew Gurley on May 30 2018

A single-page, cursory code-sample for prospective employer review. This frontend application is powered by React & Redux, written in ES6, tested with Jest and built with Brunch.

The game is a zero-player game, meaning that its evolution is determined by its initial state, requiring no further input. One interacts with the Game of Life by creating an initial configuration and observing how it evolves, or, for advanced players, by creating patterns with particular properties.

The game is a cellular automaton existing in an infinite, two-dimensional orthogonal grid of square cells, each of which exists in one of two possible states, activated or deactivated. Every cell interacts with its eight adjacent cells. At each interval in time, the following transitions occur:

  1. Any activated cell with fewer than two activated neighbors deactivates
  2. Any activated cell with two or three activated neighbors remains activated on to the next generation
  3. Any activated cell with more than three activated neighbors deactivates
  4. Any activated cell with exactly three activated neighbors becomes an activated cell
The initial pattern constitutes the seed of the system. The first generation is created by applying the above rules simultaneously to every cell in the seed. Each generation is a pure function of the preceding one. The rules continue to be applied repeatedly to create further generations.
